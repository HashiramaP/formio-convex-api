# Project Guidelines

## api.ts

The `api.ts` file is **autogenerated** by `npm run gen:api` and should **never be manually modified**.

If you need to change API types or functions:
1. Modify the source function in `convex/`
2. Run `npm run gen:api` to regenerate `api.ts`
3. Do not edit `api.ts` directly

## Admin Authorization

Use `ADMIN_WORKOS_USER_IDS`, this is simpler with WorkOS than email-based allowlists.

- Match admins by WorkOS user ID (JWT `sub` claim)
- Set via: `npx convex env set ADMIN_WORKOS_USER_IDS=user_001,user_002,...`
